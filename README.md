# Turborepo Design System for React.js with Storybook

[https://najwer23.github.io/najwer23snacks](https://najwer23.github.io/najwer23snacks)

## Self Update

```
ncu -u
```

## Yarn update

```
yarn set version latest
```

## Installation

```sh
yarn install
```

## Development

```sh
yarn start
```

## Production

```sh
yarn build
```

## Deploy to gh-pages

```sh
yarn deploy
```

## NPM packages release

```sh
yarn changeset
yarn release
```
